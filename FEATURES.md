# Weekly Planner — Planned Features

## Low Effort

- **Priority levels** — High / Medium / Low flag on tasks with visual indicators in the week grid
- **Search** — Full-text search across task titles and notes
- [Done] **Toast notifications** — Success/error feedback (currently the UI gives no confirmation)
- **Subtasks / checklists** — A list of checkboxes nested inside a task
- **Markdown notes** — Write task notes using Markdown (bold, italic, lists, code blocks, headings) with a live preview toggle

## Medium Effort

- **Recurring tasks** — Daily / weekly / monthly repeat patterns
- **Task duration / time blocks** — Add an end time to enable a day-view timeline (start time already stored)
- **Drag & drop** — Move tasks between days or reorder within a day
- **Task templates** — Save a task as a reusable template (e.g. "Weekly review")

## High Effort

- **Team / shared workspace** — Shared calendar with invite links, task assignment, and per-member views (Team model already scaffolded in schema)
- **Analytics dashboard** — Completion rates, tasks per category, streaks over time
- **Notifications / reminders** — Browser push or email alerts for upcoming tasks
